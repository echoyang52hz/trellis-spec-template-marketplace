# Trellis Spec Template Marketplace

这个仓库保存可复用的 Trellis spec 模板。模板内容沉淀自真实项目实践，但已抽离具体业务、技术栈、服务名和部署细节，只保留项目流程治理、任务生命周期、Git / MR 协作、计划单元文档和通用思考指南。

## 模板目录
